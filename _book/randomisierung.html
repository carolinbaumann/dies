8  Randomisierung

8.1 Was ist Randomisierung?

Randomisierung bezeichnet die zufällige Zuweisung von Untersuchungsteilnehmenden zu den verschiedenen Untersuchungsbedingungen einer Studie, beispielsweise einer Experimental- und einer Kontrollgruppe. Sie stellt ein zentrales Merkmal experimenteller Forschungsdesigns dar und dient dazu, systematische Unterschiede zwischen den Gruppen zu minimieren.

Durch die zufällige Gruppenzuweisung werden sowohl bekannte Merkmale (z. B. Vorwissen, Motivation oder Alter) als auch unbekannte Einflussfaktoren im Erwartungswert gleichmäßig auf die Untersuchungsgruppen verteilt. Auf diese Weise erhöht sich die Wahrscheinlichkeit, dass beobachtete Unterschiede in den Ergebnissen tatsächlich auf die untersuchte Intervention zurückzuführen sind.

Wichtig zu beachten

Randomisierung garantiert keine identischen Gruppen. Insbesondere bei kleinen Stichproben können weiterhin Unterschiede zwischen den Gruppen auftreten. Ihr entscheidender Vorteil besteht jedoch darin, dass diese Unterschiede nicht systematisch durch den Forschungsprozess erzeugt werden.

Randomisierung trägt somit wesentlich zur internen Validität einer Studie bei und bildet eine wichtige Grundlage für die Ableitung kausaler Aussagen.

8.2 Zentrale Funktionen der Randomisierung

  • Reduktion systematischer Verzerrungen (Bias): Durch zufällige Zuweisung werden systematische Fehler vermieden
  • Herstellung vergleichbarer Untersuchungsgruppen: Gruppen sind im Erwartungswert ähnlich
  • Kontrolle bekannter und unbekannter Einflussfaktoren: Auch nicht gemessene Merkmale werden ausgeglichen
  • Erhöhung der internen Validität: Kausale Schlussfolgerungen werden plausibler
  • Verbesserung der Voraussetzungen für kausale Interpretationen: Interventionseffekte lassen sich besser identifizieren

8.3 Beispiel aus der Deutschdidaktik

Bei der Evaluation einer Lesefördermaßnahme oder einer Intervention zur Schreibförderung soll durch Randomisierung sichergestellt werden, dass Unterschiede zwischen den Gruppen möglichst nicht bereits vor Beginn der Maßnahme bestehen. Dadurch können Leistungsunterschiede nach der Intervention plausibler auf die Maßnahme selbst zurückgeführt werden.

Stellen Sie sich vor: Eine Klasse erhält ein neues Lesestrategieprogramm, während eine Vergleichsklasse regulären Unterricht erhält. Ohne Randomisierung könnte die Lehrkraft unbewusst die leistungsstärkere Klasse für das Programm auswählen – die späteren Unterschiede wären dann nicht eindeutig dem Programm zuzuschreiben.

8.4 Interaktives Tool: Randomisierung durchführen

Mit diesem Tool können Sie eine einfache Randomisierung für Ihre eigene Studie simulieren.

  value: 40,
  step: 1,
  label: "Anzahl Teilnehmende"
})

viewof n_groups = Inputs.range([2, 4], {
  value: 2,
  step: 1,
  label: "Anzahl Gruppen"
})

viewof group_names = Inputs.text({
  label: "Gruppennamen (durch Komma getrennt)",
  value: "Experimentalgruppe, Kontrollgruppe",
  placeholder: "Gruppe 1, Gruppe 2, ..."
})

viewof do_randomize = Inputs.button("Randomisierung durchführen")
function randomizeParticipants(n, groups) {
  const groupList = groups.split(',').map(g => g.trim()).slice(0, n_groups);
  
  // Erstelle Array von Teilnehmer-IDs
  const participants = Array.from({length: n}, (_, i) => i + 1);
  
  // Fisher-Yates Shuffle
  for (let i = participants.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [participants[i], participants[j]] = [participants[j], participants[i]];
  }
  
  // Weise Gruppen zu
  const assignments = participants.map((id, index) => ({
    id: id,
    group: groupList[index % groupList.length]
  }));
  
  // Zähle Gruppengröße
  const groupCounts = {};
  groupList.forEach(g => groupCounts[g] = 0);
  assignments.forEach(a => groupCounts[a.group]++);
  
  return { assignments, groupCounts, groupList };
}

// Führe Randomisierung durch
randomization = {
  do_randomize; // Abhängigkeit vom Button
  return randomizeParticipants(n_participants, group_names);
}

8.6 Arten der Randomisierung

8.6.1 Einfache Randomisierung

Bei der einfachen Randomisierung wird jede Person unabhängig mit gleicher Wahrscheinlichkeit einer Gruppe zugewiesen (z. B. durch Münzwurf oder Zufallszahlengenerator).

Vorteil: Einfach durchzuführen
Nachteil: Bei kleinen Stichproben können ungleiche Gruppengrößen entstehen

8.6.2 Block-Randomisierung

Bei der Block-Randomisierung werden Teilnehmende in Blöcken randomisiert, um gleiche Gruppengrößen zu garantieren.

Beispiel: Bei 40 Personen und 2 Gruppen werden 4 Blöcke mit je 10 Personen gebildet. In jedem Block werden 5 Personen der Experimental- und 5 der Kontrollgruppe zugewiesen.

8.6.3 Stratifizierte Randomisierung

Bei der stratifizierten Randomisierung werden Teilnehmende zunächst nach wichtigen Merkmalen (z. B. Geschlecht, Alter, Vorwissen) in Schichten (Strata) eingeteilt und dann innerhalb jeder Schicht randomisiert.

Vorteil: Gewährleistet Gleichverteilung wichtiger Kovariaten

8.7 Überprüfung der Randomisierung

Nach der Randomisierung sollten die Gruppen auf Baseline-Merkmale verglichen werden, um zu überprüfen, ob die Zufallszuweisung erfolgreich war:

  • Demografische Merkmale (Alter, Geschlecht)
  • Relevante Vortestleistungen
  • Motivationale Variablen

Dies geschieht typischerweise durch:

  1. Deskriptive Statistiken (Mittelwerte, Standardabweichungen)
  2. Statistische Tests (t-Tests, Chi²-Tests) - allerdings mit Vorsicht zu interpretieren!
Zur Interpretation von Baseline-Tests

Signifikante Unterschiede bei Baseline-Vergleichen bedeuten nicht, dass die Randomisierung fehlgeschlagen ist. Bei α = 0.05 sind 5% signifikante Ergebnisse rein zufällig zu erwarten. Die Randomisierung bleibt trotzdem die beste Methode zur Gruppenzuweisung.
