7  Poweranalyse für t-Tests

7.1 Interaktive Poweranalyse

Mit diesem Tool können Sie die erforderliche Stichprobengröße für einen unabhängigen t-Test berechnen.

viewof effect_size = Inputs.range([0.1, 2.0], {
  value: 0.5, 
  step: 0.1, 
  label: "Effektgröße (Cohen's d)"
})

viewof alpha = Inputs.range([0.01, 0.15], {
  value: 0.05, 
  step: 0.01, 
  label: "Signifikanzniveau (α)"
})

viewof power = Inputs.range([0.5, 0.99], {
  value: 0.8, 
  step: 0.05, 
  label: "Gewünschte Power (1-β)"
})

viewof ratio = Inputs.range([0.5, 3.0], {
  value: 1.0, 
  step: 0.1, 
  label: "Verhältnis Gruppe 2 / Gruppe 1"
})
jstat = require("jstat@1.9.6")

function calculateSampleSize(d, alpha, power, ratio) {
  // Z-Werte für zweiseitigen Test
  const z_alpha = jstat.normal.inv(1 - alpha/2, 0, 1);
  const z_beta = jstat.normal.inv(power, 0, 1);
  
  // Formel für Stichprobengröße bei ungleichen Gruppengrößen
  const n1 = Math.pow((z_alpha + z_beta), 2) * 2 * (1 + 1/ratio) / Math.pow(d, 2);
  const n2 = n1 * ratio;
  
  return {
    n1: Math.ceil(n1),
    n2: Math.ceil(n2),
    total: Math.ceil(n1 + n2)
  };
}

sample_sizes = calculateSampleSize(effect_size, alpha, power, ratio)

7.1.1 Interpretation der Effektgröße

  • d = 0.2: Kleiner Effekt
  • d = 0.5: Mittlerer Effekt (Standard)
  • d = 0.8: Großer Effekt

7.1.2 Hintergrund

Die Poweranalyse berechnet die Mindeststichprobengröße, um einen Effekt mit der gewünschten Wahrscheinlichkeit (Power) zu entdecken.

Wichtige Begriffe: - α (Alpha): Wahrscheinlichkeit eines Fehlers 1. Art (fälschliches Ablehnen von H₀) - Power (1-β): Wahrscheinlichkeit, einen tatsächlich vorhandenen Effekt zu entdecken - Effektgröße (d): Standardisierte Differenz zwischen den Gruppen

7.2 Warum sind Poweranalysen wichtig?

Poweranalysen gehören zu den wichtigsten Schritten bei der Studienplanung. Sie helfen dabei, die Stichprobengröße so festzulegen, dass wissenschaftliche Fragestellungen mit ausreichender statistischer Sicherheit untersucht werden können.

7.2.1 Nutzen einer Poweranalyse

  • Vermeidung zu kleiner Stichproben, die vorhandene Effekte übersehen.
  • Vermeidung unnötig großer Stichproben, die Zeit, Geld und Ressourcen verschwenden.
  • Transparente Begründung der geplanten Stichprobengröße.
  • Erfüllung wissenschaftlicher und ethischer Anforderungen bei Studien.

Ohne Poweranalyse besteht die Gefahr, dass eine Studie zwar durchgeführt wird, aber keine aussagekräftigen Ergebnisse liefern kann.

7.3 Was bedeutet statistische Power?

Die statistische Power beschreibt die Wahrscheinlichkeit, einen tatsächlich existierenden Effekt als statistisch signifikant zu erkennen.

  • Power = 0,80 (80 %) bedeutet: Wenn der Effekt tatsächlich existiert, wird er in 80 % vergleichbarer Studien entdeckt.
  • β (Beta) = 1 − Power ist die Wahrscheinlichkeit eines Fehlers 2. Art.
  • Bei einer Power von 0,80 beträgt β = 0,20. In diesem Fall würden 20 % der Studien einen vorhandenen Effekt übersehen.

Typische Zielwerte:

Power Interpretation
0,70 Mindestanforderung in explorativen Studien
0,80 Häufig verwendeter Standard
0,90 Hohe Sicherheit für konfirmatorische Studien
0,95 Sehr hohe Sicherheit, erfordert meist größere Stichproben

7.4 Wie wird eine Poweranalyse berechnet?

Eine Poweranalyse basiert auf vier zentralen Größen:

  1. Effektgröße (d)
  2. Signifikanzniveau (α)
  3. Power (1−β)
  4. Stichprobengröße (N)

Sind drei dieser Größen bekannt, kann die vierte berechnet werden.

7.4.1 Zusammenhang der Parameter

  • Größere Effektgrößen benötigen weniger Teilnehmende.
  • Höhere gewünschte Power erfordert größere Stichproben.
  • Kleinere Alpha-Niveaus (z. B. 0,01 statt 0,05) erfordern größere Stichproben.
  • Ungleich große Gruppen erhöhen meist den Stichprobenbedarf.

7.4.2 Formel für den unabhängigen t-Test

Für eine vereinfachte a-priori-Poweranalyse kann die erforderliche Stichprobengröße näherungsweise mit folgender Formel berechnet werden:

[ n = ]

Dabei gilt:

  • (z_{/2}): kritischer z-Wert für das gewählte Alpha-Niveau
  • (z_{}): z-Wert der gewünschten Power
  • (d): Cohen’s d

Das interaktive Tool verwendet dieselbe Grundidee und berücksichtigt zusätzlich unterschiedliche Gruppengrößen.

7.5 Konsequenzen einer zu geringen Power

Studien mit geringer Power haben mehrere Nachteile:

  • Reale Effekte werden häufig übersehen.
  • Ergebnisse schwanken stärker zwischen Studien.
  • Gefundene Effekte werden tendenziell überschätzt.
  • Die Reproduzierbarkeit wissenschaftlicher Befunde sinkt.

Aus diesem Grund wird in vielen Fachgebieten eine Power von mindestens 0,80 empfohlen.
