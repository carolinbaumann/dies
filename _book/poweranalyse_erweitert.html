6  Wozu dient eine Poweranalyse?

Eine Poweranalyse hilft dabei, die erforderliche Stichprobengröße für eine Studie zu bestimmen. Ziel ist es, einen tatsächlich vorhandenen Effekt mit ausreichender Wahrscheinlichkeit zu entdecken.

Sie unterstützt damit eine begründete Studienplanung:

  • Zu kleine Stichproben können reale Effekte übersehen.
  • Unnötig große Stichproben verbrauchen zusätzliche Zeit und Ressourcen.
  • Die geplante Stichprobengröße wird transparent und methodisch nachvollziehbar begründet.

6.1 Zentrale Größen

Eine Poweranalyse verbindet vier Größen miteinander:

  1. Effektgröße
    Beschreibt, wie stark ein erwarteter Unterschied oder Zusammenhang ausfällt. Beim Vergleich zweier Gruppen wird häufig Cohen’s (d) verwendet.

  2. Signifikanzniveau (alpha)
    Gibt an, wie hoch die akzeptierte Wahrscheinlichkeit für einen Fehler 1. Art ist, also für das fälschliche Ablehnen der Nullhypothese.

  3. Power (1-beta)
    Beschreibt die Wahrscheinlichkeit, einen tatsächlich vorhandenen Effekt statistisch zu erkennen.

  4. Stichprobengröße (N)
    Gibt an, wie viele Personen für die Untersuchung benötigt werden.

Sind drei dieser Größen festgelegt, kann die vierte bestimmt werden.

6.2 Was bedeutet statistische Power?

Eine Power von 0,80 bedeutet: Wenn der angenommene Effekt tatsächlich existiert, wird er in 80 % vergleichbarer Studien als statistisch signifikant erkannt.

Die Gegenwahrscheinlichkeit ist der Fehler 2. Art:

[ beta = 1 - Power ]

Bei einer Power von 0,80 beträgt (beta = 0,20). In 20 % vergleichbarer Studien würde ein tatsächlich vorhandener Effekt übersehen.

Typische Zielwerte sind:

Power Einordnung
0,80 häufig verwendeter Mindeststandard
0,90 höhere Sicherheit, meist größere Stichprobe
0,95 sehr hohe Sicherheit, deutlich größerer Stichprobenbedarf
Note

Der passende Zielwert hängt vom Forschungsziel, vom Risiko eines übersehenen Effekts und von den verfügbaren Ressourcen ab. Eine Power von 0,80 ist eine häufig verwendete Konvention, aber kein allgemeingültiges Naturgesetz.

6.3 Folgen einer zu geringen Power

Studien mit geringer Power haben mehrere Probleme:

  • Tatsächlich vorhandene Effekte werden häufiger übersehen.
  • Ergebnisse fallen von Studie zu Studie instabiler aus.
  • Statistisch signifikante Effekte werden tendenziell überschätzt.
  • Die Reproduzierbarkeit der Befunde sinkt.

Die Poweranalyse sollte deshalb bereits vor der Datenerhebung erfolgen und gemeinsam mit Forschungsfrage, Design und Auswertungsstrategie geplant werden.

7.1 Beispiel: Poweranalyse t-Test

Mit diesem Tool können Sie die erforderliche Stichprobengröße für einen unabhängigen t-Test berechnen.

viewof effect_size = Inputs.range([0.1, 2.0], {
  value: 0.5, 
  step: 0.1, 
  label: "Effektgröße (Cohen's d)"
})

viewof alpha = Inputs.range([0.01, 0.15], {
  value: 0.05, 
  step: 0.01, 
  label: "Signifikanzniveau (α)"
})

viewof power = Inputs.range([0.5, 0.99], {
  value: 0.8, 
  step: 0.05, 
  label: "Gewünschte Power (1-β)"
})

viewof ratio = Inputs.range([0.5, 3.0], {
  value: 1.0, 
  step: 0.1, 
  label: "Verhältnis Gruppe 2 / Gruppe 1"
})
jstat = require("jstat@1.9.6")

function calculateSampleSize(d, alpha, power, ratio) {
  // Z-Werte für zweiseitigen Test
  const z_alpha = jstat.normal.inv(1 - alpha/2, 0, 1);
  const z_beta = jstat.normal.inv(power, 0, 1);
  
  // Formel für Stichprobengröße bei ungleichen Gruppengrößen
  const n1 = Math.pow((z_alpha + z_beta), 2) * 2 * (1 + 1/ratio) / Math.pow(d, 2);
  const n2 = n1 * ratio;
  
  return {
    n1: Math.ceil(n1),
    n2: Math.ceil(n2),
    total: Math.ceil(n1 + n2)
  };
}

sample_sizes = calculateSampleSize(effect_size, alpha, power, ratio)

7.1.1 Interpretation der Effektgröße

  • d = 0.2: Kleiner Effekt
  • d = 0.5: Mittlerer Effekt
  • d = 0.8: Großer Effekt
